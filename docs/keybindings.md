# GridLock Keybindings

GridLock supports global keybindings, including Vim-style chorded sequences.

## Standard Debugging
* **F5**: Run / Continue
* **F10**: Step Over
* **F11**: Step Into

## Window Navigation (Vim-style Chords)
* **Ctrl+W, H**: Focus Editor
* **Ctrl+W, L**: Focus Project Tree

## Actions
* **Alt+R**: Run Target (launch new session)
* **Alt+B**: Toggle Breakpoint on current line
