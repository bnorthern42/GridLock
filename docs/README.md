# GridLock User Documentation

Welcome to the official User Guide for GridLock, a Wayland-native, Qt6-based graphical debugger tailored for MPI applications. GridLock is designed to accelerate the debugging workflow for HPC researchers and C++ engineers by providing deep visibility into distributed state, intuitive configuration management, and seamless integration with high-performance computing environments.

## Table of Contents

### 1. User Guide
Core workflows, interface paradigms, and IDE configuration.
* [Interface & Workflows](user_guide/interface_workflows.md) - Learn how to navigate the UI, inspect variables across MPI ranks, and evaluate C++ expressions.
* [Configuration & Management](user_guide/configuration.md) - Master keyboard shortcuts, session persistence, and offline reference manual integration.

### 2. HPC Integration
Deploying and debugging at scale.
* [SLURM & Spack Integration](hpc_integration/slurm_and_spack.md) - Configure remote SSH execution, manage SLURM batch jobs, and handle Spack environment modules.
